# OCum-Addiction
An expansion of OCum providing addiction buffs/debuffs from swallowing a lot of cum. Similar to the cum addiction in SL Survival. This is still under development and has very few features at the moment. I didn't include a project file since all my sources were local.

### Requirements
[SKSE](https://skse.silverlock.org/)

[Sky UI](https://www.nexusmods.com/skyrimspecialedition/mods/12604) - [Source](https://github.com/schlangster/skyui)

[PapyrusUtil SE](https://www.nexusmods.com/skyrimspecialedition/mods/13048)

[ConsoleUtilSSE](https://www.nexusmods.com/skyrimspecialedition/mods/24858)

[OStim](https://www.github.com/Sairion350/OStim)

[OSA - Skyrim Ascendency Engine](https://www.nexusmods.com/skyrimspecialedition/mods/17217)
